## ✅ قابلیت‌های تکمیل‌شده

### فاز 0 — زیرساخت پایه

- [x] Monorepo با Turborepo و pnpm.
- [x] NestJS برای API و Next.js برای وب.
- [x] PostgreSQL و Redis در مسیر توسعه محلی.
- [x] TypeORM با entity loading.

### فاز 1 — احراز هویت

- [x] مدل کاربر و نقش‌های Student/Teacher/Admin.
- [x] ثبت‌نام، OTP توسعه، ورود، refresh، logout و بازیابی رمز.
- [x] JWT Guard و Role Guard.
- [x] Rate limit مبتنی بر Redis برای مسیرهای حساس.

### فاز 2 — دوره و محتوا

- [x] Entityهای Course، CourseSection، Lesson، Enrollment و CourseReview.
- [x] URLهای دستی ویدیو برای کیفیت‌های 720p/1080p/480p بدون پردازش سروری.
- [x] APIهای عمومی دوره‌ها و APIهای مدرس/دانشجو.
- [x] کش Redis برای لیست و جزئیات دوره‌های منتشرشده.
- [x] کنترل دسترسی محتوای درس بر اساس enrollment، مالکیت مدرس یا free preview.

### فاز 3 — فروش و پرداخت

- [x] Entityهای Order، OrderItem و PaymentTransaction.
- [x] ساخت سفارش و جلوگیری از خرید تکراری.
- [x] درخواست پرداخت MVP و callback/verify با الگوی زرین‌پال‌مانند.
- [x] ثبت تراکنش و خطاهای پرداخت.
- [x] ایجاد Enrollment بعد از پرداخت موفق.

### فاز 4 — فرانت‌اند MVP

- [x] صفحه اصلی، لیست دوره‌ها و صفحه جزئیات دوره.
- [x] صفحات ورود، ثبت‌نام، تایید OTP و بازیابی رمز.
- [x] پنل دانشجو و مسیر تماشای درس با پلیر HTML5.
- [x] پنل مدرس/ادمین برای فرم‌های پایه دوره، جلسه، کیفیت و سفارش‌ها.
