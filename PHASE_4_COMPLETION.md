# فاز 4 — پنل‌های MVP فرانت‌اند 🧩

**تاریخ تکمیل:** ۱۴۰۵/۰۳/۲۵ | **وضعیت:** ✅ تکمیل شده برای MVP

---

## خلاصه

فاز 4 یک رابط کاربری MVP برای فروش و مصرف دوره‌های ضبط‌شده اضافه کرد. صفحات با داده نمونه محلی ساخته شده‌اند تا مسیر محصول، خرید، وضعیت دسترسی، تماشای درس و مدیریت پایه مدرس بدون وابستگی به API production قابل نمایش باشد.

## موارد تکمیل‌شده

- صفحه اصلی با معرفی Talkotopia LMS و دوره‌های منتخب.
- صفحه لیست دوره‌ها با جستجو و فیلترهای پایه نمایشی.
- صفحه جزئیات دوره با سرفصل‌ها، قیمت، CTA خرید و وضعیت دسترسی.
- صفحات احراز هویت MVP: ورود، ثبت‌نام، تایید OTP و بازیابی رمز.
- پنل دانشجو برای دوره‌های خریداری‌شده، دسترسی قفل‌شده و ثبت نظر.
- اتصال مسیر تماشای درس به پلیر HTML5 موجود با انتخاب کیفیت و دانلود مجاز.
- پنل مدرس/ادمین برای ایجاد/ویرایش دوره، مدیریت URLهای کیفیت 720p/1080p/480p، وضعیت انتشار و مشاهده پایه سفارش‌ها.

## نکته MVP

فرم‌ها در این فاز UI-first هستند و برای اتصال نهایی باید به APIهای احراز هویت، دوره‌ها، سفارش‌ها و پرداخت وصل شوند. ساختار مسیرها و فیلدها مطابق APIهای فعلی انتخاب شده‌اند.
